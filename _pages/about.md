# About FuturisticSEC
--------------------

![Me](http://gravatar.com/avatar/6ea61db1afd0fe022af40cbe4f70e19e?s=200){: .profile .pull-right }

FuturisticSEC is multi-talented web programming shop operated by **Umoru Lucky Zakari** in Lagos, NG.
